# qinVideo

[查看项目](https://github.com/Qinmei/qinVideo)

这是一款基于 egg.js + mongodb + redis 的视频 CMS 系统, 以美学设计为先, 希望能够给个人站长更多选择, 也可以根据后台提供的 API 自行定制前端页面, 目前正处于 V2 版本, 还需要继续打磨, 详细情况可进入项目查看

### 主要功能

- 视频的播放, 支持苹果 CMS 采集接口
- 专栏文章的显示
- 用户注册登录
- 商品购买
- 评论以及弹幕发送
